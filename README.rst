                                                        -------------------------------------------
                                                                    QR Code Generator
                                                        -------------------------------------------

This is a QR Code Generator which generates a qr code for the provided link.

For this program to run, install a module "qrcode" by the following command:

``pip3 install qrcode``

Now run the python file by:

``python3 code12.py``

It will generate a ``qrcode.png`` file.
